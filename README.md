# TechOps Provider Support Guide

An internal self-service web app for LocumTenens.com Recruiters and CRs to look up common provider technical issues without needing to contact the TechOps team.

## What It Does

Recruiters and CRs can search by keyword or browse by category to find step-by-step resolutions for the most common provider issues. Each issue card includes the problem description, common causes, numbered resolution steps, and any important warnings or tips.

## Features

- **Keyword search** — searches across issue titles, descriptions, causes, steps, and notes with highlighted matches
- **Category filtering** — browse issues by topic using the sidebar
- **Expandable cards** — click any issue to reveal the full resolution
- **Escalation contact** — TechOps helpdesk info and ticket template always visible in the sidebar
- **No dependencies** — single HTML file, works offline, no server required

## Categories Covered

| Category | Topics |
|---|---|
| Login & Account | Password reset, locked accounts |
| Documents | Upload/open failures, Document Library explained |
| Email & Invites | Missing emails, safe sender setup |
| Credentialing | Accessing the app, re-credentialing, sync delays |
| Axuall | Identity verification errors, auto-populate vs. manual fields |
| Mobile & Browser | Mobile compatibility errors, browser troubleshooting |
| Portal & Tabs | Missing Credentials tab, verification confusion |

## Usage

Open `index.html` in any browser — no installation or setup needed.
